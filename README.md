# Kairo.AI
The Proactive Personal AI Assistant.

## Architecture
- **Backend**: FastAPI with a multi-agent system.
- **Frontend**: Next.js App Router with premium dashboard design.

## Features
- **Proactive Suggestions**: Agents analyze your habits to suggest orders/rides.
- **Voice UI**: Real-time voice interaction with visual feedback.
- **Insights Dashboard**: Analytics on spending, habits, and trends.
- **Smart Calendar**: cron-inspired calendar with auto-scheduling.
