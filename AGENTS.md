# Agent Architecture

## Decision Agent
Analyzes multiple service providers (Zomato, Swiggy, Blinkit, Uber, Ola) to find the best value for the user.

## Data Agent
Simulates live API data for testing purposes, including traffic, availability, and pricing.

## Memory Agent
Tracks user habits and preferences to provide personalized, proactive suggestions.

## Trigger Agent
Monitors time and context to trigger suggestions at the right moment.
